### 如何更新

一如既往，您可以使用以下任一命令更新您的机器人：

#### docker-compose

```bash
docker-compose pull
docker-compose up -d
```

#### 通过安装脚本进行安装

```
# 退出虚拟环境并运行
./setup.sh --update
```

#### 纯本机安装

```
git pull
pip install -U -r requirements.txt
```
